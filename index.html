Mentor Mentee Form Automation Tool

Download ZIP See Source Code

Follow these instructions to run the file:

  1. Download the ZIP file by clicking the "Download ZIP" button above.
  2. Extract the contents of the ZIP file to a folder on your computer.
  3. Open the extracted folder and locate the main application file.
  4. Double-click the application file to launch the tool.
  5. Upon launching, wait for upto 1 minute while the application loads.
  6. A GUI window will be launched prompting you to enter the registration numbers.
  7. Enter the registration numbers for all the students you wish to retireve data, with one reg. no. on each line.
  8. Click on Submit
  9. On Clicking on Submit a new browser window opens, please login with your SLCM Credentials
  10. Wait patiently as the data is retireved for all the students.
  11. The data for each individual student will be stored in a folder named by the registration no.
  12. Each folder will contain individual PDF files for each semester of the student
  13. It will also contain an excel file which can be opened and saved as PDF or printed, to get the Student Details for that student.
  14. All downloaded files files within a particular folder (for a student) are named in order cooresponding to their page number in the form.

Requirements:

  1. A stable internet connection is required to access and interact with the online platform.
  2. Google Chrome browser is required for running this tool. Make sure you have Google Chrome installed on your computer. (Preferabelly version 107)
  3. If you are not sure about your Google Chrome version, follow these steps to check it:
    1. Open Google Chrome.
    2. Click on the three vertical dots in the top-right corner to open the menu.
    3. Hover over "Help" in the menu, and a sub-menu will appear. Select "About Google Chrome."
    4. A new tab will open, displaying the current version of Google Chrome.
  4. If your Google Chrome version does not match the recommended version, you can download and install the appropriate version of Chromedriver that matches your Google Chrome version from the following link: Chromedriver Download or from Chromium Download
  5. Replace the downloaded chromedriver.exe file downloaded instead of the cooresponding file present in the unzipped folder.
